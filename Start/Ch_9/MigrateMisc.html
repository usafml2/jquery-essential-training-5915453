Miscellaneous Migration Features

jQuery's CSS handling features have modern browser quivalents. Elements support a classList property to manipulate CSS classes.

addClass() equivalent is classList.add()

removeClass() equivalent is classList.remove()

toggleClass() equivalent is classList.toggle()

Updating the DOM also has standard methods.

jQuery's text() method equivalent is the textContent property

Standard DOM functions are used instead of $("<elem>") to create new content
